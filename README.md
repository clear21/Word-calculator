# Word-calculator

# Overview
英単語の足し算・引き算を電卓感覚で楽しめます。

# Description
自然言語処理Word2Vecの魅力の一つとして、単語のベクトル化による単語の足し算・引き算があります。  
当アプリは、英単語の足し算・引き算が自由にでき、Word2Vecの魅力を体感できます。
